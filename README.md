# Project 05 Calculators

This project has 3 calculators made with simple HTML, CSS and JavaScript.

## Files
- `index.html`
- `style.css` (one common CSS file)
- `script.js` (one common JS file)
- `interest-calculator/index.html`
- `bmi-calculator/index.html`
- `calorie-calculator/index.html`

## Calculators
1. **Simple Interest**
   - Input: principal, rate, time
   - Output: interest and total amount
   - Formula: `(P × R × T) / 100`

2. **BMI**
   - Input: height (cm), weight (kg)
   - Output: BMI value and category
   - Formula: `weight / (height in meter)^2`

3. **Calorie**
   - Input: age, gender, weight, height, activity
   - Output: daily calories
   - Formula: Mifflin-St Jeor + activity multiplier

## Notes
